## 📌 About The Project
Switching between backend, mobile, and scripting languages can be painful. Syntax is forgotten, features evolve. This project is a massive, carefully curated reference guide comparing **Swift, Kotlin, Python, and Go** side-by-side. 

It covers everything from basic syntax to modern 2026 paradigms:
- **Concurrency:** `async/await`, `Actors`, `Coroutines`, `Goroutines`, Python's `Free-Threading` & `JIT`.
- **Architecture:** Dependency Injection, Repository patterns, Builders.
- **Advanced:** Regex, Serialization, Cryptography (AES-GCM, JWT), File I/O.

## ✨ Features
- **Code Tabs:** Click on a language tab to instantly switch the code snippet without scrolling.

## 🛠 How to run locally (For Contributors)
Want to add a new section or fix a typo? Running the site locally is easy.

1. Clone the repository:
   ```bash
   git clone https://github.com/silencepeaceoff/swift-kotlin-python-guide.git
   cd swift-kotlin-python-guide
   ```
2. Install MkDocs and required plugins:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the local development server:
   ```bash
   mkdocs serve
   ```
4. Open `http://127.0.0.1:8000` in your browser. The site will auto-reload when you edit any `.md` file in the `docs/` folder!
