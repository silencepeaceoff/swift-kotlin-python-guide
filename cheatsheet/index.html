    Краткая справочная таблица синтаксиса для быстрого сравнения.

    Объявления и типы🔗

    Концепция Swift 6 Kotlin 2.x Python 3.13 Go 1.24+
    Переменная var x = 5 var x = 5 x = 5 var x = 5 / x := 5
    Константа let x = 5 val x = 5 x = 5 (соглашение X = 5) const x = 5
    Тип аннотация let x: Int = 5 val x: Int = 5 x: int = 5 var x int = 5
    Nullable var x: Int? = nil var x: Int? = null x: int \| None = None var x *int = nil (указатель)
    Unwrap x!, x ?? 0, if let x x!!, x ?: 0, x?.let {} x or 0 (truthy) if x != nil { *x }
    Приведение типа x as! Int, x as? Int x as Int, x as? Int int(x), isinstance(x, int) int(x), x.(int) (для interface)
    Строковая интерп. "Hello \(name)" "Hello $name" f"Hello {name}" fmt.Sprintf("Hello %s", name)
    Multiline string """...""" """...""" """...""" `...`

    Коллекции🔗

    Концепция Swift 6 Kotlin 2.x Python 3.13 Go 1.24+
    Массив [1, 2, 3] listOf(1, 2, 3) [1, 2, 3] []int{1, 2, 3} (slice)
    Мутабельный var arr = [1,2] mutableListOf(1,2) arr = [1,2] arr = append(arr, 3)
    Словарь ["a": 1, "b": 2] mapOf("a" to 1) {"a": 1, "b": 2} map[string]int{"a": 1}
    Множество Set([1, 2, 3]) setOf(1, 2, 3) {1, 2, 3} map[int]struct{}{1: {}}
    Tuple (1, "a") Pair(1, "a") (1, "a") Множественный возврат функцией
    Диапазон 1...5, 1..<5 1..5, 1 until 5 range(1, 6) for i := range 5 (Go 1.22+)

    Управление потоком🔗

    Концепция Swift 6 Kotlin 2.x Python 3.13 Go 1.24+
    If if x > 0 { } if (x > 0) { } if x > 0: if x > 0 { }
    If-expr let y = x > 0 ? "+" : "-" val y = if (x > 0) "+" else "-" y = "+" if x > 0 else "-" Нет (писать if-else явно)
    Switch switch x { case .a: ... } when (x) { is A -> ... } match x: case A(): ... switch x { case A: ... }
    For for i in 0..<5 { } for (i in 0 until 5) { } for i in range(5): for i := 0; i < 5; i++ { }
    For-each arr.forEach { } arr.forEach { } for x in arr: for i, v := range arr { }
    While while x > 0 { } while (x > 0) { } while x > 0: for x > 0 { }
    Guard/require guard x > 0 else { return } require(x > 0) assert x > 0 if x <= 0 { return }

    Функции🔗

    Концепция Swift 6 Kotlin 2.x Python 3.13 Go 1.24+
    Объявление func add(_ a: Int, _ b: Int) -> Int fun add(a: Int, b: Int): Int def add(a: int, b: int) -> int: func add(a, b int) int { }
    Лямбда { (a, b) in a + b } { a, b -> a + b } lambda a, b: a + b func(a, b int) int { return a + b }
    Аргумент по умол. func f(x: Int = 0) fun f(x: Int = 0) def f(x: int = 0): Нет (нужны паттерны builder/options)
    Variadic func f(_ n: Int...) fun f(vararg n: Int) def f(*args): func f(n ...int)
    Named args f(x: 1, y: 2) f(x = 1, y = 2) f(x=1, y=2) Нет (передаются 3 struct Options{})
    Extension extension Int { func sq() } fun Int.sq() Нет func (i MyInt) sq() (только локальные типы)

    ООП🔗

    Концепция Swift 6 Kotlin 2.x Python 3.13 Go 1.24+
    Класс class Dog { } class Dog { } class Dog: type Dog struct { } (value-тип!)
    Наследование class Dog: Animal { } class Dog : Animal() { } class Dog(Animal): Нет (только композиция Embedded struct)
    Интерфейс protocol P { } interface P { } class P(Protocol): type P interface { } (Duck Typing!)
    Data class — (struct + Equatable) data class Point(val x: Int) @dataclass class Point: type Point struct { X int }
    Enum enum E { case a, b } enum class E { A, B } class E(Enum): type E int; const (A E = iota)
    Sealed enum (с assoc values) sealed class / sealed interface Union[A, B] (TypeVar) Чаще интерфейс + приватный метод
    Singleton static let shared object Singleton Модуль / __new__ var shared = &Singleton{}; sync.Once
    Struct/Value struct S { } @JvmInline value class NamedTuple type S struct { } (Всегда передаётся по значению, если не *S)

    Concurrency🔗

    Концепция Swift 6 Kotlin 2.x Python 3.13 Go 1.24+
    Async func func f() async throws -> T suspend fun f(): T async def f() -> T: Обычная функция (go f())
    Await try await f() f() (в suspend контексте) await f() Блокировка (<-ch или wg.Wait())
    Параллельно async let a = f() async { f() } asyncio.gather() go f() (горутина)
    Task group withTaskGroup coroutineScope { } asyncio.TaskGroup() errgroup.Group / sync.WaitGroup
    Stream AsyncStream Flow async for Базовые каналы (chan T)
    Actor/Lock actor A { } Mutex() asyncio.Lock() sync.Mutex / каналы
    Главный поток @MainActor Dispatchers.Main loop.call_soon В Go нет UI-очереди, OS Runtime

    Обработка ошибок🔗

    Концепция Swift 6 Kotlin 2.x Python 3.13 Go 1.24+
    Выкинуть throw MyError.x throw MyException() raise MyError() return nil, errors.New("err")
    Поймать do { try f() } catch { } try { f() } catch () { } try: f() except: if err != nil { return err }
    Optional ошибка try? f() runCatching { f() } Нет аналога if err != nil
    Result Result<T, Error> Result<T> (stdlib) Pydantic Result Возврат кортежа (T, error)
    Panic/Crash fatalError() throw RuntimeException() raise Exception() panic("crash") / recover()
